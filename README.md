# Parsewave-Game-Jam-2026: Space Defense

## How to Play
- Use your **mouse** to move the rocket left and right.
- **Click** your mouse to fire lasers at the incoming asteroids.
- Avoid letting asteroids pass the base (you have 5 lives).
- The game gets faster as your score increases!

## What was built in the 24-hour window
- Developed a web-based space defense game from scratch.
- Implemented custom canvas-based rendering for a pixel-style rocket with flame animations.
- Engineered a dynamic difficulty system where asteroid spawn rates and speeds scale with the player's score.
- Created "Space Junk" asteroid shapes using procedural drawing logic.

## AI Tools Used
- Used AI-agent assistance (Gemini) to plan the game loop, debug collision logic, and generate the procedural pixel-art rocket and jagged asteroid functions.
